# Study Resources Sharing Portal

This is a simple web application for sharing and rating study resources.

## Features

-   **Upload Resources:** Users can upload files (e.g., PDFs, DOCX, PPTX) as study resources.
-   **Download Resources:** Users can download the uploaded resources.
-   **Rate Resources:** Users can rate the resources on a 5-star scale.
-   **User Authentication:** Users can register and log in to the application.

## Technologies Used

-   **Backend:**
    -   Java
    -   Spring Boot
    -   Spring Security
    -   Spring Data JPA
    -   H2 (in-memory database)
-   **Frontend:**
    -   Thymeleaf
    -   Bootstrap CSS
-   **Build Tool:**
    -   Maven

## How to Run

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/varsha8906/study-resource-rating.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd study-resource-rating
    ```
3.  **Run the application using Maven:**
    ```bash
    ./mvnw spring-boot:run
    ```
4.  **Access the application:**
    Open your web browser and go to `http://localhost:8080`.
